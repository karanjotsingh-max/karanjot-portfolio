# Karanjot Singh - Portfolio

A minimalist portfolio website showcasing my work as a Fullstack Software Developer, built with Next.js and Tailwind CSS.

## About

Personal portfolio featuring my professional experience, projects, and technical skills. Currently working as a Fullstack Software Developer at Government of Alberta, specializing in .NET, React, and GenAI/RAG applications.

## Features

- **Minimalist Design** - Clean typography with Geist font and strategic use of whitespace
- **Responsive Layout** - Mobile-first design that works on all devices
- **Theme Toggle** - Seamless dark/light mode switching
- **Smooth Animations** - Subtle scroll-triggered animations and hover effects

## Tech Stack

- [Next.js 15](https://nextjs.org/) - React framework
- [TypeScript](https://www.typescriptlang.org/) - Type safety
- [Tailwind CSS](https://tailwindcss.com/) - Styling
- [next-themes](https://github.com/pacocoursey/next-themes) - Theme management

## Getting Started

```bash
# Clone the repository
git clone https://github.com/karanjotsingh-max/My-Portfolio.git

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the portfolio.

## Project Structure

```
├── app/
│   ├── layout.tsx      # Root layout with theme provider
│   ├── page.tsx        # Main portfolio page
│   ├── globals.css     # Global styles and CSS variables
├── components/
│   └── theme-provider.tsx  # Dark/light theme provider
├── public/
│   └── images/         # Project images
└── lib/
    └── utils.ts        # Utility functions
```

## Customization

- Update personal information in `app/page.tsx`
- Modify colors and design tokens in `app/globals.css`
- Add or remove sections as needed
